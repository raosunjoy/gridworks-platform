# TradeMate Partner Portal - Component Library

Welcome to the **TradeMate Partner Portal** component library! This Storybook instance contains all the UI components, pages, and patterns used throughout our partner portal application.

## 🏗️ Architecture Overview

Our portal is built with modern technologies:

- **Next.js 14** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **Zustand** for state management
- **NextAuth.js** for authentication
- **Prisma** for database management

## 📚 Component Categories

### Authentication Components
- **SignInForm**: Complete authentication form with OAuth support
- **SignUpForm**: User registration with validation
- **ErrorBoundary**: Error handling and recovery

### Dashboard Components
- **SelfHealingDashboard**: System health monitoring
- **AdminDashboard**: Administrative interface
- **DeveloperPortal**: API documentation and tools

### UI Components
- **ToastProvider**: Notification system
- **LoadingSpinner**: Loading states
- **Modal**: Dialog components

## 🎨 Design System

### Colors
Our design system uses a carefully crafted color palette:

## 🔧 Development Guidelines

### Component Standards
1. **TypeScript**: All components must be fully typed
2. **Props Interface**: Define clear prop interfaces
3. **Default Props**: Provide sensible defaults
4. **Accessibility**: Follow WCAG guidelines
5. **Responsive**: Mobile-first design approach

### Testing
- **Unit Tests**: Jest + React Testing Library
- **Visual Tests**: Storybook visual regression
- **E2E Tests**: Playwright for user journeys

### State Management
- **Local State**: React hooks (`useState`, `useReducer`)
- **Global State**: Zustand stores
- **Server State**: React Query
- **Form State**: React Hook Form

## 🎯 Usage Examples

### Basic Component Usage
```tsx
import { SignInForm } from '@/components/auth/SignInForm';

function LoginPage() {
  const handleSubmit = (data) => {
    // Handle form submission
  };

  return (
    <SignInForm 
      onSubmit={handleSubmit}
      isLoading={false}
    />
  );
}
```

### State Management Example
```tsx
import { useAuthStore } from '@/store/auth';

function UserProfile() {
  const { user, logout } = useAuthStore();
  
  return (
    <div>
      <h1>Welcome, {user?.name}</h1>
      <button onClick={logout}>Logout</button>
    </div>
  );
}
```
